# 컴퓨터 프로그래밍 파이썬 학생 성적 및 출결 관리 프로그램

## 프로젝트 소개
이 프로젝트는 파이썬의 핵심 자료구조(List, Dictionary)를 활용하여 학생들의 성적과 출결 데이터를 터미널에서 관리할 수 있도록 만든 프로그램입니다. 
모듈(Module)을 분리하여 코드의 재사용성을 높이고, 독립적인 가상환경(venv) 위에서 작동하도록 설계되었습니다.

## 기획 의도
파이썬의 핵심 자료구조인 리스트(List)와 딕셔너리(Dict)가 실제 환경에서 어떻게 데이터를 묶고 처리하는지 직접 구현해 보며 그 원리를 이해하고자 기획되었습니다.

## 프로젝트 목적
1.**구조적 코딩 설계**: 모든 코드를 한곳에 몰아넣지 않고, 기능과 실행을 모듈로 분리하여 코드의 가독성과 재사용성을 높입니다.
2.**독립적 개발 환경**: 가상환경(venv)을 세팅하고 필요한 외부 패키지(pip)만 분리해서 관리하는 방법을 체화합니다.
3.직관적인 UI : 터미널 창에서도 tabulate 라이브러리를 활용해 데이터를 깔끔한 표 형태로 만들어, 효율적인 결과물을 만듭니다.

## 주요 기능
1.학생 등록: 고유 학번과 이름을 통해 신규 학생을 등록합니다.
2.성적 입력: 학생별로 다양한 과목의 점수를 개별적으로 입력하고 저장합니다.
3.출결 기록: 출석 상태(O, X)를 순서대로 누적하여 기록합니다.
4.데이터 조회: tabulate 라이브러리를 활용해 터미널에서 데이터를 표 형태로 확인할 수 있습니다.

## 파일 소개
- `main.py`: 데이터를 입력하고 화면에 표를 출력하는 실행용 메인 스크립트
- `tools.py`: 학생 데이터를 담는 딕셔너리와 관리 함수들이 정의된 파일

## 실행 방법 
1. 프로젝트를 컴퓨터로 다운로드합니다.
2. 터미널을 열고 해당 폴더로 이동한 뒤, 가상환경을 생성하고 활성화합니다.
   python -m venv myenv
   myenv\Scripts\activate  # Windows  기준
3. 터미널에서 pip install tabulate 로 tabulate을 다운로드 합니다.
4. main.py와 tools.py를 추가한 뒤에 실행합니다.
